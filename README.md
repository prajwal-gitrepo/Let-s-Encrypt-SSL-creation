# Let-s-Encrypt-SSL-creation

## Introduction
- Use case: We needed (BYOC) certificate for our domain to validate in AFD domain validation.
- Why Let's encrypt: We have other CA like digiCert but the are not free, So we can use let's encrypt for our case.
